# Tic Tac Toe

The classic game with Spring boot and Vue.js under the hood.

## Getting Started

### Prerequisite

```
JDK 8 or higher installed with JAVA_HOME configured appropriately
```

### Compiling and running

1. Open a new terminal window.
2. Navigate to the project's folder.
3. Execute the following command:

```
./mvnw spring-boot:run
```
(On Windows, you need to use the mvnw.cmd)

4. The running application will be available at:

```
http://localhost:8080/tictactoe/index.html
```
